# UtsavBalar1231 - dotfiles

Personal dotfiles for my Arch Linux and Ubuntu systems.

The dotfiles are managed using my personal [dotfiles sync tool](https://github.com/sync-dotfiles-rs)

## Installation

### Arch Linux

```bash
./setup.sh --arch
```

### Ubuntu / Debian

```bash
./setup.sh --ubuntu
```

or

```bash
./setup.sh --debian
```

## Sync dotfiles

```bash
./sync-dotfiles-rs -F
```
